# PMM-QM/MM

Reference implementation of the PMM-QM/MM method.

This repository contains:
- A Fortran 90 code (`PMM-QMMM.f90`) for PMM-QM/MM calculations
- A Python tool (`pmm-qmmm_input_generator.py`) for automated generation of PMM and PMM-QM/MM input files
- Example input and output files for the `pmm-qmmm_input_generator.py` program

Please refer to the individual README files for detailed instructions
(`README_PMM-QMMM.txt` and `README_pmm-qmmm_input_generator.txt`).
